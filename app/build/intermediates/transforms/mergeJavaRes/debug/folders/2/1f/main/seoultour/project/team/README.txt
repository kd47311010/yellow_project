[2012년 3월 팀 프로젝트]


<주제> : [서울 여행] - 서울 여행 정보, 가이드




<메인 메뉴> : 테마여행, 내위치, 편의정보

	테마여행
 		- 고궁, 문화, 전시, 전경, 쇼핑 		
 
	내위치
 		- 카메라와 방향센서를 이용하여 증강 현실을 구현
  
	편의정보 
 		- 정보 사이트 링크  


<어플리케이션 실행 절차>

1. 실행 후 서버로부터 db 파일을 다운로드

2. db 파일 구성 : 테마, 위도, 경도, 장소 이름, 이미지링크, 홈페이지 링크, 간단한 설명, 긴 설명

3. db 파일을 이용하여 
   
   테마여행에서는 맵뷰에서 위도 경도를 연결하여 오버레이로 테마별로 보여줌
   
   내위치에서는 증강현실을 구현하여 보여줌
   
   
 
 <파일별 설명>
 
SeoulTourActivity.java
 
 : 메인 액티비티 

   메인 화면을 보여주고 로딩 스레드를 실행시킴
   
   
   
Loading.java
 
 : 로딩 스레드
 
   서버에 접속하여 db 파일을 다운로드 받음
   
   다운로드 완료시 메인 액티비티를 종료하고 메인메뉴 액티비티로 전환됨
   
   
   
MainMenuActivity.java

 : 메인메뉴 액티비티
 
   테마여행, 내위치, 편의정보를 선택할 수 있게 해주는 액티비티
   
   터치시 해당 액티비티로 전환
   
   
   
   
ThemeMeneuActivity.java

 : 테마여행의 하위 메뉴액티비티
 
   해당 테마 터치시 지도를 보여주는 액티비티가 실행됨 
   
     
   
SeoulMapActivity.java

 : 구글맵을 이용하여 지도를 보여주는 액티비티
 
   터치된 테마의 값을 꺼내와서 db를 열어 위도, 경도의 좌표값, 정보, 링크 등을 읽어
   
   오버레이 아이템들을 만들고 오버레이를 생성하여 등록함
   
   현재위치를 찾음, 버튼을 이용한 현재 위치로 이동 기능 
   
   
   
SeoulMapOverlay.java
 
 : 등록된 오버레이 아이템을 관리함
 
   오버레이 터치시 이미지와 정보가 포함된 레이아웃을 띄움
   
   레이아웃 터치시 정보 액티비티로 전환됨   
   
 

CameraActivity.java

 : 내위치 터치시 전환되는 액티비티
 
   카메라의 서페이스 뷰(카메라를 처리하는 전용 뷰)를 등록하고
   
   서페이스 뷰 위에 보여질 커스텀 뷰(카메라 오버레이 뷰)를 등록함
   
   현재위치를 찾고 찾은 현재위치를 카메라 오버레이 뷰에 알려줌
   
   이때 GPS, 네트워크를 이용하여 현재 위치를 찾음
   
   GPS가 이용가능하면 GPS 이용
   
   불가능하면 네트워크를 이용하여 현재위치를 찾음
   
   둘다 불가능하면 위치 정보 설정으로 이동을 물어보는 액티비티로 이동
   

CameraPreview.java

 : 카메라의 서페이스 뷰
 
   카메라를 등록하고 관리함
   
   
CameraOverlayView.java

  : 커스텀 뷰(View를 상속받고 onDraw를 구현하여 뷰를 원하는대로 보여지게 함)를 사용
  
   일정시간(방향 센서값이 전달될때)마다 onDraw를 다시 실행되게 하여 뷰를 새로 그림
   
   현재위치 좌표와 db에서 읽어들인 장소의 좌표를 이용하여 동쪽을 기준으로 한 각도를 계산
   
   방향센서값중 x 좌표(지구를 위에서 아래로 바라보았을때 좌표 - 나침판 값)값을 위에서
   
   계산한 값에 더하여 현재 안드로이드폰이 향하고 있는 방향을 적용하여 장소를 보여줌
   
   터치를 입력 받아서 보여지는 범위(km)조정, 보여질 장소 테마 선택, 아이템 터치시
   
   간단한 정보 팝업, 팝업 터치시 정보 페이지로 이동기능
   
 
LocationSettingRequese.java

 : 위치정보를 설정으로 이동을 물어보는 액티비티


  
DBHandler.java

 : 다운로드하여 저장된 db 파일을 관리함
 
  선택된 테마별로 db파일에서 레코드들을 읽어 리스트를 만들고 리턴함
  
  이때 레코드 마다 DBRecord 객체를 생성하여 리스트로 만듬
  
  
DBRecord.java

 : db 각 하나의 레코드 클래스
 
  db에서 읽은 레코드에 대응 됨
  
 
InfoActivity.java

 정보를 보여주는 액티비티, db에서 선택된 아이템을 찾아서 사진과, 설명, 링크 홈페이지를 보여줌
